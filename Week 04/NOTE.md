# 学习笔记

## 知识点

#### charCodeAt() 
获取字符串指定位置的 Unicode 编码
##### 语法
```
string.charCodeAt(index)
```
* index，必需。表示字符串中某个位置的数字，即字符在字符串中的下标。


#### fromCharCode() 
将 Unicode 编码转为一个字符:
##### 语法
```
string.charCodeAt(number0, number1, number2... )
```
* number0, number1, number2... ,  必须，一个或多个 Unicode 编码
